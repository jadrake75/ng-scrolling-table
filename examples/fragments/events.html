TableEvents

constant in ng-scrolling-table Provides constants for the emitted events by various directives and services within the ng-scrolling-table module.

Events

ConstantString ValueDetails
insertRows insert-rows Emitted when new rows are inserted into the table and detected by the table services
deleteRows delete-rows Emitted when new rows are deleted from the table and detected by the table services. Two data fields will be included:
tableId - the table that emitted the event
deleted - the count of the number of rows deleted.
changeVisibility change-visibility Emitted when a column is either shown or hidden by either programmatic or user actions
selection select Emitted when a row(s) is selected. The reference id of the row item will be returned as an array of all selected items in the data field
clearSelection clear-selection Emitted when all selections have been cleared.
