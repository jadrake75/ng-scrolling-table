table-columns-resizable

directive in ng-scrolling-table Adds the ability to define columns as resizeable within the table. By default all non-fixed width columns (those not marked as column-fixed="true") will be configured to support resizing.

Applicable Tags

  • TABLE

column-fixed

directive in ng-scrolling-table Designates a column as being fixed, preventing the resizing directive from adding resize handle support for the column.

Applicable Tags

  • TH

Arguments

ParamTypeDetails
column-fixed String If set to "true" will mark the column as fixed width and not support resizing

Examples

Resizable Table with First Column Fixed

In the following example, the country table header is fixed at a width of 45% of the total table width, however all of the other columns are adjustable.
